Digital Driver Dashboard

Undergraduate Final Year Project - Skills Exercised: C++, CAN, Raspberry Pi, SolidWorks

Developed a high-frequency (60Hz) real-time driver dashboard that decodes CAN bus signals, restructures telemetry data and displays key metrics for driver feedback and rapid diagnostics.

  • Implemented a multithreaded C++ program, reading from an existing vehicle CAN bus, to feed the UI at 60Hz.
  • Built an on-vehicle UI optimized for readability under motion and low-latency updates.
  • Created offline test harnesses to simulate CAN traffic for robust validation away from the vehicle.

Impact: Enabled live driver feedback and rapid fault diagnosis during testing sessions; shortened debugging cycles.

MAN24 Wiring Harness

Formula Student IC Electrical Sub-team Leader - Skills Exercised: Electrical System Design, Fault Finding

Designed a full club-spec wiring harness for the 2024 FS car running a Yamaha R6 engine.

  • Established standards for connectors, wire coverings & power distribution
  • Implemented data logging to replay test sessions and trace faults.
  • Planned, documented and built the harness from scratch, accounting for appropriate lengths, future servicability and abrasion resistance.

Impact: Passed all electrical inspection at FSUK 2024 and ran engine for a number of future tuning sessions

High Voltage Battery Pack

Formula Student Electrical Safety Officer - Skills Exercised: HV Safe Work, Electrical System Design

Built the high voltage battery for the University of Manchester's first student built electric car. Collaborated with two others to construct, wire and test safety circuits, whilst following safe working practices for high voltage.
